# Backend Services

Services backend pour la gestion du quiz TikTok Live.

## Structure

- `services/tiktok/` - Service de connexion TikTok Live
- `services/quiz/` - Logique métier du quiz
- `services/tts/` - Text-to-Speech
- `services/database/` - Accès base de données
- `lib/` - Utilitaires (Redis, Supabase clients)
- `types/` - Types TypeScript partagés

## API Routes

Les routes API Next.js sont dans `app/api/` à la racine du projet.
